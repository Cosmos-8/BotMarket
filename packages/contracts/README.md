# BotMarket Contracts

Base contracts for BotMarket bot registry.

## Setup

```bash
# Install Foundry if not already installed
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Install dependencies
forge install
```

## Build

```bash
forge build
```

## Test

```bash
forge test
```

## Deploy

See deployment scripts in `script/` directory.
